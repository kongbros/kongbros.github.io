# 메모리 게임

## 게임 개요
짝 맞추기 게임을 만든다.
다양한 종류의 패를 2장씩 만들어서 랜덤하게 섞은 후 뒤집어 놓는다.
사용자는 패를 두 장씩 열어가면서 같은 패를 찾는 게임이다.
두 장 고르고 일치하면 유저의 성공, 해당 패는 유저의 것이 되고, 일치하지 않으면 다시 뒤집는다.
적은 시간 동안, 적은 횟수로 성공할수록 좋은 점수를 얻는다.

## 게임 플로우
시작 버튼을 누르면 게임이 시작된다. 첫 3초간 패가 열려있는 상태로 보여진다.
3초가 지나면 패가 모두 뒤집히면서 뒷면만 노출한다.
유저는 그때부터 뒤집기를 시작한다.
마지막 패까지 모두 맞히면 게임이 종료된다.

## 게임 패 구성
characters.png 파일을 열어서 5x6 의 그리드로 캐릭터들을 잘라서 사용한다. 총 30개의 캐릭터 중 랜덤으로 10개를 선택하여 10쌍(20장)의 카드로 게임을 구성한다.

## 구현
static web 에서도 동작이 가능하도록 html 로 구현한다.

## 게임 방법
1. **게임 시작**: index.html 파일을 브라우저에서 열어 게임을 시작합니다.
2. **카드 뒤집기**: 카드를 클릭하여 뒤집고 캐릭터를 확인합니다.
3. **짝 맞추기**: 두 장의 카드를 뒤집어 같은 캐릭터를 찾습니다.
4. **게임 목표**: 모든 카드의 짝을 맞춰 게임을 완료합니다.

## 게임 특징
- **타이머**: 게임 시작부터 완료까지의 시간을 측정합니다.
- **움직임 카운트**: 카드를 뒤집은 횟수를 기록합니다.
- **애니메이션 효과**: 카드 뒤집기, 매치 성공 시 부드러운 애니메이션을 제공합니다.
- **반응형 디자인**: 모바일 기기에서도 즐겁게 플레이할 수 있습니다.
- **다시 시작**: 언제든지 게임을 초기화하고 새로 시작할 수 있습니다.

## 파일 구조
```
memory-game/
├── index.html      # 메인 HTML 파일
├── style.css       # 스타일시트
├── script.js       # 게임 로직
├── characters.png  # 캐릭터 이미지
└── readme.md       # 게임 설명
```

## 기술 스택
- **HTML5**: 시맨틱 마크업 사용
- **CSS3**: 그리드 레이아웃, 애니메이션, 반응형 디자인
- **Vanilla JavaScript**: ES6+ 클래스 기반 객체 지향 프로그래밍
