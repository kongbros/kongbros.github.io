# 직소 퍼즐 게임 (Jigsaw Puzzle Game)

## 1. 개요 (Overview)

이 프로젝트는 웹 기반의 직소 퍼즐 게임입니다. 사용자가 직접 이미지를 업로드하면 해당 이미지가 4x5 크기의 슬라이딩 퍼즐로 변환됩니다. 퍼즐의 마지막 조각이 제거된 상태에서 나머지 조각들이 무작위로 섞이며, 사용자는 조각들을 밀어서 원래 위치로 맞춰 퍼즐을 완성하는 것이 목표입니다.

## 2. 주요 기능 (Features)

- **이미지 업로드**: 로컬 기기에서 퍼즐로 사용할 이미지를 선택하여 업로드할 수 있습니다.
- **난이도 선택**: 1단계(가장 쉬움)부터 10단계(가장 어려움)까지 난이도를 선택할 수 있으며, 단계에 따라 퍼즐 조각이 섞이는 정도가 결정됩니다.
- **4x5 그리드**: 업로드된 이미지는 4x5 격자(총 20조각)로 나뉩니다.
- **대화형 퍼즐**:
    - 오른쪽 하단의 마지막 조각을 제거하여 빈 공간을 만듭니다.
    - 나머지 19개의 조각은 무작위로 섞입니다.
    - 사용자는 빈 공간에 인접한 조각을 클릭하여 이동시킬 수 있습니다.
- **게임 정보 제공**:
    - **타이머**: 게임 시작과 동시에 경과 시간을 초 단위로 표시합니다.
    - **이동 횟수(Move Counter)**: 사용자가 조각을 움직인 횟수를 추적하여 표시합니다.
    - **이미지 미리보기**: 참고용으로 원본 전체 이미지를 작게 보여줍니다.
- **승리 조건**: 모든 퍼즐 조각이 원래의 올바른 위치로 돌아가면 게임에서 승리합니다.
- **점수 산정**: 승리 시 완료 시간과 이동 횟수를 바탕으로 점수를 계산합니다. 시간이 짧고 이동 횟수가 적을수록 더 높은 점수를 얻습니다.
- **사용자 경험(UX)**:
    - 게임 승리 후 "다시 플레이하기" 옵션을 제공합니다.
    - 조각을 움직일 때 효과음이 재생됩니다.
    - 조각이 올바른 위치에 배치되거나 인접할 때 반짝임(sparkles)이나 흔들림(shakes) 같은 시각적 효과를 주어 사용자를 가이드합니다.

## 3. 기술 스택 (Technical Stack)

- **프론트엔드**: HTML, CSS, JavaScript
- **라이브러리**: 외부 라이브러리나 프레임워크를 사용하지 않은 순수 자바스크립트(Vanilla JS) 프로젝트입니다.

## 4. 파일 구조 (File Structure)


```

.
├── index.html       # 게임의 구조를 정의하는 메인 HTML 파일
├── style.css        # 게임의 외관과 레이아웃을 담당하는 스타일시트
├── script.js        # 게임 기능 및 로직을 포함하는 자바스크립트 파일
├── GEMINI.md        # 프로젝트에 대한 상세 개요 문서 (본 문서)
└── README.md        # 프로젝트 메인 리드미 파일

```

## 5. 게임 방법 (How to Play)

1. 프로젝트 파일을 클론(Clone)하거나 다운로드합니다.
2. 최신 웹 브라우저(Chrome, Firefox 등)에서 `index.html` 파일을 엽니다.
3. 드롭다운 메뉴에서 난이도를 선택합니다.
4. "이미지 선택" 버튼을 클릭하여 컴퓨터에 있는 이미지 파일을 불러옵니다.
5. 게임이 자동으로 시작됩니다. 퍼즐 조각을 클릭하여 원래 위치로 맞춰보세요!
