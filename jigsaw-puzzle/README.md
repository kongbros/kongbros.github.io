# Jigsaw Puzzle Game

이 프로젝트는 사용자가 이미지를 업로드하여 클래식한 4x5 슬라이딩 퍼즐을 즐길 수 있는 웹 기반 직소 퍼즐 게임입니다.

## 1. 주요 기능 및 사양

### 요구사항
- **이미지 업로드**: 사용자는 자신의 컴퓨터에서 원하는 이미지를 선택하여 퍼즐 게임을 생성할 수 있습니다.
- **4x5 그리드**: 업로드된 이미지는 4x5, 총 20개의 조각으로 나뉘며, 마지막 한 조각은 비워둡니다.
- **조각 섞기 및 이동**: 게임 시작 시 19개의 조각이 무작위로 섞이며, 사용자는 빈칸과 인접한 조각을 클릭하여 위치를 바꿀 수 있습니다.
- **난이도 조절**: 1(쉬움)부터 10(어려움)까지 난이도를 선택할 수 있으며, 난이도에 따라 조각이 섞이는 횟수가 달라집니다.
- **게임 클리어**: 모든 조각을 원래 이미지의 위치에 맞추면 게임이 종료됩니다.
- **점수 시스템**: 게임이 완료되면, 소요 시간과 이동 횟수를 바탕으로 점수가 계산됩니다. 시간이 짧고 이동 횟수가 적을수록 높은 점수를 얻습니다.

### UI 구성
- **게임 보드**: 4x5 그리드의 퍼즐판이 중앙에 위치합니다.
- **원본 이미지 미리보기**: 우측 사이드바에서 완성된 이미지의 축소판을 확인할 수 있습니다.
- **게임 정보**: 사이드바 상단에 실시간으로 **경과 시간**과 **이동 횟수**가 표시됩니다.

### 특수 효과 (타격감)
- **✨ 스파클 효과**: 두 개의 퍼즐 조각이 서로 인접하고, 두 조각 모두 자신의 최종 위치에 올바르게 놓이면 그 경계선에 반짝이는 효과가 나타납니다. 보드에 올바르게 맞춰진 모든 쌍을 시각적으로 확인할 수 있습니다.
- **⚡️ 지진 효과**: 사용자가 **방금 움직인 조각**이 올바른 위치로 이동했고, 인접한 조각 또한 올바른 위치에 있을 경우, **방금 움직인 조각에만** 짧은 지진 효과가 발생하여 정확한 이동임을 알려줍니다.

## 2. 게임 방법 및 팁

### 게임 방법
1.  게임 페이지에 접속하면 가장 먼저 난이도를 선택합니다. (기본값: 5)
2.  `이미지 업로드` 버튼을 클릭하여 퍼즐로 만들고 싶은 이미지 파일을 선택합니다.
3.  이미지를 선택하는 즉시 게임이 자동으로 시작됩니다.
4.  빈칸 주변에 있는 조각을 클릭하여 빈칸으로 이동시킵니다.
5.  사이드바의 미리보기 이미지를 참고하여 모든 조각을 제자리에 맞춥니다.
6.  퍼즐을 완성하면 최종 점수가 표시되고, `다시하기` 버튼으로 새 게임을 시작할 수 있습니다.

### 게임 팁
- **가장자리부터 공략하세요**: 실제 퍼즐처럼, 이미지의 가장자리에 해당하는 조각들을 먼저 맞추면 전체적인 윤곽을 잡기 수월합니다.
- **스파클 효과를 잘 활용하세요**: 조각들 사이에 나타나는 반짝임은 두 조각이 모두 제자리에 있다는 확실한 증거입니다. 이 효과를 기준으로 나머지 조각들의 위치를 유추해 보세요.
- **지진 효과로 확신을 얻으세요**: 방금 옮긴 조각이 흔들렸다면, 그 수는 '정답'입니다. 계속해서 다음 조각을 맞춰나가세요.
- **한 줄씩 완성하기**: 위에서부터 한 줄씩, 혹은 왼쪽에서부터 한 열씩 순서대로 맞춰나가는 것도 좋은 전략입니다.
